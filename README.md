# kcl-lang.io

Source for KCL site <https://kcl-lang.io>.

Powered by [Docusaurus 2](https://docusaurus.io/).

## Update Dependencies

```bash
npm install
```

## Build

```bash
npm run start
```

Open http://localhost:3000 in the browser.

## Build with i18n

```bash
npm run build
npx http-server ./build
```

Open http://localhost:8080 in the browser.
